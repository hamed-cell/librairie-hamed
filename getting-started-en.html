Overview

Formidable is a powerful library for simplifying form management in your web applications. It provides a suite of reusable components that make it easy to create custom forms while maintaining performance and accessibility.

Motivation

The development of Formidable was motivated by the need to standardize form handling.

Component Definition

The BasicForm component is defined as a function that takes formName as a prop to customize the title of the form displayed to the user.

function BasicForm({ formName }) {
  const handleSubmit = (e) => {
    e.preventDefault(); // Prevents the default form submission behavior
  };

  return (
    <div className="basic-form">
      <p>{formName}</p>
      ...
    </div>
  );
}

Submission Handling Logic

handleSubmit is a function that is called when the submission button is clicked. It uses e.preventDefault() to stop the page from reloading, which is essential for not losing state during form submission.

Building Form Fields

Each form field uses a specific component that encapsulates the logic needed to collect and validate data:

Each input is designed to be reusable and easily integrable into other parts of the application, increasing consistency and reducing code redundancy.

Props Validation

The BasicForm component uses PropTypes to ensure that the formName prop is passed correctly as a string, reinforcing the component's robustness.

BasicForm.propTypes = {
  formName: PropTypes.string, // Validates that formName is indeed a string
};

BasicForm Component

The BasicForm is a functional React component that uses individual input components to collect user data. It is structured to be easily integrable and extensible:

import PropTypes from "prop-types";
import { InputTypeText, InputTypeEmail, InputTypePassword, InputTypeCheckbox, InputTypeButton } from "../IndividualsInputs";
import "../styles/Forms.css";

function BasicForm({ formName }) {
    const handleSubmit = (e) => {
        e.preventDefault(); // Prevents page reload
    };

    return (
        <div className="basic-form">
            <p>{formName}</p>
            <label> Name </label>
            <InputTypeText placeholder="Enter your name" />
            <label> First Name </label>
            <InputTypeText placeholder="Enter your first name" />
            <label> Age </label>
            <InputTypeText placeholder="Enter your age" type="number" min="0" max="100" />
            <label> Email </label>
            <InputTypeEmail />
            <label> Password </label>
            <InputTypePassword />
            <InputTypeCheckbox />
            <label> I agree to transmit my information </label>
            <InputTypeButton onClick={handleSubmit}>Create My Account</InputTypeButton>
        </div>
    );
}

BasicForm.propTypes = {
    formName: PropTypes.string
};

export default BasicForm;

This component uses PropTypes to ensure that the properties passed to the form adhere to the expected type, thus increasing the component's robustness.

Installation

To integrate Formidable into your project:

git clone https://github.com/Messaoudiam/Form-idable.git
cd Form-idable
npm install
npm run dev
