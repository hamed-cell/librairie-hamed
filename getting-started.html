Vue d'ensemble

Formidable est une bibliothèque puissante pour simplifier la gestion de formulaires dans vos applications web. Elle fournit une suite de composants réutilisables qui facilitent la création de formulaires personnalisés tout en maintenant la performance et l'accessibilité.

Motivation

Le développement de Formidable a été motivé par la nécessité de standardiser le traitement des formulaires .

Définition du Composant

Le composant BasicForm est défini comme une fonction prenant formName en prop pour personnaliser le titre du formulaire affiché à l'utilisateur.

function BasicForm({ formName }) {
  const handleSubmit = (e) => {
    e.preventDefault(); // Empêche le comportement par défaut de soumission du formulaire
  };

  return (
    <div className="basic-form">
      <p>{formName}</p>
      ...
    </div>
  );
}

Logique de Gestion des Soumissions

handleSubmit est une fonction qui est appelée lors du clic sur le bouton de soumission. Elle utilise e.preventDefault() pour stopper le rechargement de la page, ce qui est essentiel pour ne pas perdre l'état lors de la soumission du formulaire.

Construction des Champs du Formulaire

Chaque champ du formulaire utilise un composant spécifique qui encapsule la logique nécessaire pour collecter et valider les données :

<label> Nom </label>
<InputTypeText placeholder="Entrez votre nom" />

<label> Prénom </label>
<InputTypeText placeholder="Entrez votre prénom" />

<label> Age </label>
<InputTypeText placeholder="Entrez votre age" type="number" min="0" max="100" />

<label> Email </label>
<InputTypeEmail />

<label> Mot de passe </label>
<InputTypePassword />

<InputTypeCheckbox />
<label> J'accepte de transmettre mes informations </label>

<InputTypeButton onClick={handleSubmit}>
  Je crée mon compte
</InputTypeButton>

Chaque input est conçu pour être réutilisable et facilement intégrable dans d'autres parties de l'application, augmentant ainsi la cohérence et réduisant la redondance du code.

Validation des Props

Le composant BasicForm utilise PropTypes pour s'assurer que la prop formName est passée correctement comme une chaîne de caractères, renforçant la robustesse du composant.

BasicForm.propTypes = {
  formName: PropTypes.string, // Valide que formName est bien une chaîne de caractères
};

Composant BasicForm

Le BasicForm est un composant fonctionnel en React qui utilise des composants d'input individuels pour collecter les données utilisateur. Il est structuré pour être facilement intégrable et extensible :

import PropTypes from "prop-types";
import { InputTypeText, InputTypeEmail, InputTypePassword, InputTypeCheckbox, InputTypeButton } from "../IndividualsInputs";
import "../styles/Forms.css";

function BasicForm({ formName }) {
    const handleSubmit = (e) => {
        e.preventDefault(); // Empêche le rechargement de la page
    };

    return (
        <div className="basic-form">
            <p>{formName}</p>
            <label> Nom </label>
            <InputTypeText placeholder="Entrez votre nom" />
            <label> Prénom </label>
            <InputTypeText placeholder="Entrez votre prénom" />
            <label> Âge </label>
            <InputTypeText placeholder="Entrez votre âge" type="number" min="0" max="100" />
            <label> Email </label>
            <InputTypeEmail />
            <label> Mot de passe </label>
            <InputTypePassword />
            <InputTypeCheckbox />
            <label> J'accepte de transmettre mes informations </label>
            <InputTypeButton onClick={handleSubmit}>Je crée mon compte</InputTypeButton>
        </div>
    );
}

BasicForm.propTypes = {
    formName: PropTypes.string, // Valide que formName est bien une chaîne de caractères
};

export default BasicForm;

Ce composant utilise PropTypes pour s'assurer que les propriétés passées au formulaire respectent le type attendu, augmentant ainsi la robustesse du composant.

Composants d'Input Individuels

Chaque input est encapsulé dans son propre composant, permettant une réutilisation et une personnalisation aisées. Par exemple, InputTypeButton est défini comme suit :

import PropTypes from "prop-types";

function InputTypeButton({ children, ...restProps }) {
    return ;
}

InputTypeButton.propTypes = {
    children: PropTypes.node.isRequired, // Assure que le bouton contient des éléments enfants valides
};

export default InputTypeButton;

Installation

Pour intégrer Formidable dans votre projet :

npm i @shark-man/react-form
npm run dev
