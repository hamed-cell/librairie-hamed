Installation of Formidable Library

Step 1: Installing the Library

To get started, you need to install the "Formidable" library into your project. This can be done using npm, the Node.js package manager. Run the following command in your terminal at the root of your project:

npm i @shark-man/react-form

This command downloads and installs the latest version of the "Formidable" library into your project, along with all its dependencies.

Step 2: Importing Components

Once the library is installed, you can import the necessary components into your React application. For example, if you want to use the BasicForm component, you can import it into your JavaScript or TypeScript file like this:

import BasicForm from 'shark-man/react-form';

Make sure to adjust the import path according to the actual location of the file in your project.

Step 3: Using Components

Once you have imported the components, you can use them like any other React component in your application. For example, to use the BasicForm component, you can include it in the rendering of your parent component like this:


function App() {
    return (
        <div className="App">
            <h1>My Application with Formidable</h1>
            <BasicForm formName="My Form" />
        </div>
    );
}

export default App;

Make sure to pass any necessary props to the components, such as formName in this example.

Step 4: Customization

You can also customize the components by passing them additional props according to your needs. For example, you can specify specific options for form fields or event handlers for user actions.
