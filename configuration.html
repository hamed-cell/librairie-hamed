Installation de la bibliothèque Formidable

Étape 1: Installation de la bibliothèque

Pour commencer, vous devez installer la bibliothèque "Formidable" dans votre projet. Cela peut se faire en utilisant npm, le gestionnaire de paquets Node.js. Exécutez la commande suivante dans votre terminal à la racine de votre projet :

npm i @shark-man/react-form

Cette commande télécharge et installe la dernière version de la bibliothèque "Formidable" dans votre projet, ainsi que toutes ses dépendances.

Étape 2: Importation des composants

Une fois la bibliothèque installée, vous pouvez importer les composants nécessaires dans votre application React. Par exemple, si vous souhaitez utiliser le composant BasicForm, vous pouvez l'importer dans votre fichier JavaScript ou TypeScript comme ceci :

import BasicForm from '@shark-man/react-form';

Assurez-vous d'adapter le chemin d'importation en fonction de l'emplacement réel du fichier dans votre projet.

Étape 3: Utilisation des composants

Une fois que vous avez importé les composants, vous pouvez les utiliser comme n'importe quel autre composant React dans votre application. Par exemple, pour utiliser le composant BasicForm, vous pouvez l'inclure dans le rendu de votre composant parent comme ceci :

function App() {
            return (
                <div className="App">
                    <h1>Mon Application avec Formidable</h1>
                    <BasicForm formName="Mon Formulaire" />
                </div>
            );
        }

        export default App;

Assurez-vous de passer toutes les props nécessaires aux composants, telles que formName dans cet exemple.

Étape 4: Personnalisation

Vous pouvez également personnaliser les composants en leur passant des props supplémentaires selon vos besoins. Par exemple, vous pouvez spécifier des options spécifiques pour les champs de formulaire ou des gestionnaires d'événements pour les actions utilisateur.
